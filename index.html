Utils for Log4-serie

About

Some utilities for Log4-serie data-loggers by Tektyte.

These have been created by a user of Log4-serie and are not affiliated with Tektyte in any way.

Utilities

  • Log Filter - filter and format Log4-serie CSV logs
    ALPHA VERSION - can be buggy and features might change without notice

Requirements

I mainly test these with Firefox in Linux, however these should work with any recent web browser.

(If not, bug reports are welcome.)

Using locally

To use these locally, just download or clone the GitHub repository and open the files in web browser.

I am creating everything manually for now, instead of using any code-generators, so everything will work directly.
