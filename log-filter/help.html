Help - Log Filter

Usage

  1. Paste input data to Input-field.
    • Output snippet will auto-update once Input-field loses focus.
  2. Change settings to your liking.
    • Output snippet will auto-update.
  3. Press 'Show All' button to see full output.
  4. Copy output from Output-field.
ps. 'Update snippet' button shouldn't be needed at all, it's just there while testing that all auto-updating works.

Input format

  • Input format is CSV log format of Log4.USB data-logger.
  • Each line has three values separated by commas:
    1. Unix Timestamp in integer milliseconds
    2. Measured current in integer microamps
    3. Measured voltage in integer millivolts
  • Spaces and empty lines are ignored.
